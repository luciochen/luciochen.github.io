Peeknest:app redesign

Overview

Peeknest is an organzing app from ios AppStore to hold all your home items and help you organize them. I redesigned user interface of item page to make them communicate clearly and add workflows to expand the function of organizing.

Context

  • Type: Course (Interface Design) project (Personal)
  • Role: Interface design, User research
  • Tools: Figma, Photoshop

Objective

Since the user interface contains some problematic visual elements and a bad information hierarchy, which influence the experience and slow down efficiency when organizing items on this app. I tried to redesign a clean and functional interface with a better visual hierarchy to make it easier to use.

Research and Testing

Conducted user tests with 3 persons.The workflow testing includes:

  • 1. Input one item by taking a photo, put it into one list
  • 2. Create a new list to hold items
  • 3. Go to the item page, edit the quantity of the item and delete the item from one of the belonging lists.

Result: Users showed a fluent performance when using the app. However, they pointed out that the information is cluttered, some visual elements are misleading. For example, the delete button is shown in a irrelevant icon. Besides, when users input more and more item, they feel hard to find the items input in a certain time.

Solution

Redesign the item page to make it clean and organized. Add a page of items overview with a filter by date added or by value.

Reflection

The project was trying to enhance the usability on the item aspect of this app. The redesign of the layout works well, the information is easier to be found than the old one. However, the critic reveals the lack of integration of the item page and the function of sorting all items. In the future projects, I should consider more about the integration of functions and the transition between functions.
