Web design solution to engage store members.

Tool library showcase

OVERVIEW

In an Interaction Design course project, we are required to collaborate with a local business find out their problem using user research and design a feasible solution to solve it.

Our final design is an online platform to enhance the store engagement by encouraging members to practice more DIY project.

Context

Course project: Interaction Design Methods (Spring 2020)

Group with Caleb, T., Dawood, S., and Fatima, I.

Role

UI design, User research, Graphic design

Tools

Illustrator, Photoshop, Figma

Vancouver Tool Library, images from: https://www.instagram.com/p/B_lb68CB3dP/

CLIENT

The Vancouver tool library is a non-profit organization that lends out physical tools. It’s a community for people who loves tools and DIY projects.

RESEARCH

During the research, I conveyed in-person interviews with 4 volunteers and a few store members. As a result, the research indicates the problem that many members are not experienced in doing DIY projects with tools.

User journey map highlight

DESIGN FOCUS

After the research, we directed our design focus to enhance new and amateur member’s tool literacy through workshop engagement and projects.

Based on members' active engagement with the Tool library website, we finally chose the solution of a project showcase on the website. It provides the members a platform to practice using tools when replicating the projects following DIY instructions.

Conclusion

Working closely with Vancouver Tool Library, I practiced user research in studying a real business and understand the problem from the users’ perspective. The manager of the store likes the design and indicate the design can be used as a good reference to develop a new section on their website.

Reflection

However, the design on the organization’s home page is not very explicit and hard to get noticed. Therefore, It would be better to run with an in-store promotion to attract store members.
