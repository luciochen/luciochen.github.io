Overview

In this project, I designed and developed the website for an online grocery store with my teammate. Our goal is to build an online shop that provides a smooth shopping experience from adding items to the cart to checking out. After the web design, we built the website using HTML/CSS/Javascript.

Web Development

My teammate and I used GitHub to share codes and distribute development tasks.

To get started, we created the basics of the website with the method of “Atomic Design”, which is creating the codes for the repeatable elements first and applying them to different pages. Then, my main responsibility was to create the home page, product page, and the pages for buying items.

Conclusion

This project was a good practice for creating a website from scratch for a client. Implementing the “Atomic Design“ in the by creating styleguide and reusable codes enhanced the efficiency of the later works.

Also, I designed the major part of the website shopping experience, crafted every steps that is required to the user interaction.

With frequent communication, my teammate and I managed to build all the fundamental functions of this website. We made efforts on the responsiveness and accessibility as well.

Reflection

In peer review, my colleagues suggested to add more information on the product detail page, so the website can provide customers adequate product information to trust the grocery product and purchase.

I believe the use of colours can still be improved. For example, on the Home page, the combination of those strong colours makes the visual hierarchy not ideal. Besides, the lack of space causes the website looks clutter on the mobile size.
