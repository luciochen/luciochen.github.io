The Online Presence Of A Branding Event

Brandstorm 2020

CONTEXT

Individual Project, November 2020

Teammate contribution:
Event Branding – Dalya F.
Feedback – David W.

ROLE

Web Design, Web Development (WordPress)

SOFTWARES

Figma, WordPress

PROJECT LINK http://www.sfusma.org/brandstorm/

Overview

Brandstorm 2020 is a branding event hosted by SFU SMA (Student Marketing Association) and an energy recycling company called Moment Energy. This event is a competition for students to create suitable branding that fits the brand identity of Moment Energy.

My job as a designer in SMA was to create a website that allows participants to view the event information and check the final results.

Goals Breakdown

Since this website targets university students, I focused on presenting the existing information in a visually appealing way. Then, I broke down the goals to:

  • Make it consistent with the event branding: young, energetic, and professional.
  • Apply a bold layout and aesthetically appearing visual elements to encourage the audience to dive deeper into the content.
  • Implement a design that is easy for the audience to locate desired content, navigate through sections, and have an enjoyable reading experience.

Inspirations

With the project requirement and the branding in mind, I collected a moodboard that communicates the visual design direction. These beautiful designs gave me inspiration for the use of color, typography, and layouts.

Moodboard

Design Process

To get started, I broke down the information into 4 sections.

  • Introduce the event and why people should join.
  • Event details
  • Partner company and the judges
  • Event schedule

2. Wireframes

In the feedback round, the wireframes got a positive response from our team lead. He suggested to maintain the layouts with some changes in the “Event Timeline” section since the “boxing” art direction doesn’t fit the branding. Also, there are some minor issues in element spacing. I implemented this feedback in the final design mockup.

Style guide
(created by my teammate Dalya)

3. Mockup Design

By using these 3 brand colors in a playful way, I created distinct visual elements and interesting layouts. For example, a gradient background with these colors was used to highlight the “Event Details“ section.

Besides, each section has a outstanding but cohesive visual identity, which makes it easy for the audience to navigate through different sections and locate information.

Web Development

Unlike the standard web coding technique (HTML/CSS), WordPress comes with many constraints when replicating the design. For example, WordPress only supports limited columns, so I needed to change the grid system to adapt to the change of layouts. And some stylings like the ones in the “Event Schedule” and “Judges” section are hard to replicate in this WordPress. Then, I decided to use a static image in the background instead.

Link to the web page: http://www.sfusma.org/brandstorm/

Due to the request from the event client, the responsibility of the web page had been changed from “Viewing the event details” to “Checking the winners and recap the whole event”. But I still managed to address this change in the content and design modification.

Conclusion

Overall, the Brandstorm 2020 website has a strong visual presentation and an engaging reading flow. The visual style attracted young competitors who favor fresh and energetic design.

In this project process, a good mood board set me a good start. It communicated the desired style and inspired me to explore these bold design variations creatively. Also, planning the information structure well made the web content flows smoothly.

In the development phase, I practiced the basics of using WordPress and adjusted the website using customized CSS. Most importantly, I learned to bridge the gap between the ideal design and the actual web-building by acknowledging the constraints of the dev tool and adjusting the design actively to make it feasible for the development.
