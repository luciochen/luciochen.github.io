The Online Presence Of A Branding Event

Brandstorm 2020

CONTEXT

Individual Project, November 2020

Teammate contribution:

Feedback – David W.

ROLE

Web Design, Web Development (Wordpress)

SOFTWARES

Figma, WordPress

PROJECT LINK http://www.sfusma.org/brandstorm/

Overview

Brandstorm is a branding event hosted by SFU SMA (Student Marketing Association) with Moment Energy. The goal of this event is to encourage participants create the appropriate branding for Moment Energy.

My job as a designer in SMA is to create a website that allows participants to view the event details and check the final results.

Goals Breakdown

Since this project requires only one web page to display all the information, I focused on creating a stylized design with an easy navigation to keep the audience interested. So I breakdown this goal to:

  • Make it consistent with the event branding: young, energetic, and professional.
  • Apply a bold layout and visually-attracting elements to encourage the audience to dive deeper in the content.
  • Implement a design that is easy for the audience to locate desired content, navigate through sections, and have an enjoyable reading experience.

Inspirations

With the project requirement and the branding in mind, I collected a moodboard that communicates the design direction. These beautiful designs gave me inspirations on the use of colour, typography, and bold layouts.

Moodboard

Design Process

To get started, I broke down the information into 4 sections. I implemented distinctive elements to keep the read interest.

  • Introduce the event and why people should join.
  • The Event details
  • The partner company and the judges
  • The event schedule

3. Mockup Design

Designing with the 3 brand colours in a playful but cohesive way, the web pages creates a visual interest and serve for the information hierarchy. For example, I use a bold gradient background in the “Event Details“ section as it’s the area I want the audience to pay attention on.

Web Development

Unlike the usual web coding (HTML/CSS), WordPress comes with many constraints in replicating the design. For example, it only provides limited columns, so I needed to change the design grid system to adapt to WordPress. And stylings like the one in the “Event Schedule” and “Judges” section are hard to replicate. Then, I decided to use a static image to compromise.

Link to the web page: http://www.sfusma.org/brandstorm/

Then, since our design team receive the project request late, the Brandstorm website planned to launch in the middle of the event competition. Therefore, the responsibility of the web page transit from “Viewing the event details” to “Checking the winners and recap the whole event”. But I still managed to address this change in the content and design modification.

Conclusion

Overall, the Brandstorm 2020 website has a strong visual presentation and an engaging reading flow. The visual style attracts young competitors who favour fresh and energetic design.

In this project process, a good mood board sets me a good start. It communicates the desired style, and inspires me to explore these bold designs variations creatively. Also, planning the information architecture well makes the web content flows smoothly and be more interesting.

In the development phrase, I learned the basics of using WordPress and adjusting the website using customized CSS. Most importantly, I learned to bridge the gap between the ideal design and the actual web-building by acknowledging the constraints of the dev tool, and adjust the design actively to make it feasible for the development.
