App redesign to fix problematic interactions

Peeknest: UI redesign

OVERVIEW

Peeknest is an organizing app from ios AppStore to hold all your home items and help you organize them. I redesigned the item page of the interface to create a better hierarchy and fixed problematic visual elements. Besides, I integrated a function of filtering all items with existing elements to let users manage items and browse them smoothly.

Context

Individual project, Course: Interface Design (Fall 2019)

Role

UI design, User research, User testing

Tools

Figma, Photoshop

OBJECTIVE

Since the user interface contains some problematic visual elements and a bad information hierarchy. These influence the experience and slow down efficiency when organizing items on this app. I redesign a clean and functional interface with a better visual hierarchy to make it easier to use.

RESERACH & TESTING

In the beginning, I Conducted user testing with 2 experienced IOS users. The workflow testing includes:

  • Input one item by taking a photo. Put it into one list
  • Create a new list to hold items
  • Go to the item page, edit the quantity of the item, and delete the item from one of the belonging lists.

Result: Users showed a fluent performance when using the app. However, they pointed out that the information is cluttered, some visual elements are misleading. For example, the delete button is shown in an irrelevant outlook. Besides, when users input more and more items, they feel hard to find the items' input at a certain time.

Pain points of the interface

Design process

Redesigned the item page to make it looks clean and organized. Added a page of items overview with a filter by date added or by value.

Conclusion

The project was trying to enhance the usability of the item aspect of this app. The redesign of the layout works well, the information is easier to be found than the old one. Also, the problematic icons got fixed, which are no longer misleading.

Reflection

The critic from the previous interviewees revealed the lack of integration of the item page and the function of sorting all items. To refine this project, I should consider more about the transition between old and new functions, and make the new functions easier to be found.
